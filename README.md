# Mines Demo: GitHub + Cloudflare Pages + Telegram Bot + D1

В проект добавлена логика демо-воронки и админка Telegram-бота. KV не используется: все игроки, блокировки, клики и статистика хранятся в Cloudflare D1 через binding `DB`.

## Что реализовано

1. После 3–5 сыгранных раундов игрок блокируется, появляется окно перехода на партнёрский сайт.
2. Если демо-счёт `$10` закончился до/на 5-й игре, окно появляется сразу.
3. После появления окна повторно играть нельзя: поле и кнопки блокируются, доступна только кнопка перехода.
4. После клика по партнёрской ссылке и возврата в Telegram WebApp показывается сообщение: «Игру можно продолжить на сайте» и кнопка перехода.
5. `/admin` в Telegram-боте показывает:
   - всего игроков;
   - игроков за 24 часа;
   - сколько перешли по ссылке;
   - сколько увидели окно, но не перешли;
   - кнопку обновления;
   - кнопку «Дожим»;в
   - кнопку помощи по сбросу игрока.
6. Дожим отправляет сообщение всем Telegram-пользователям из базы, которые ещё не нажали партнёрскую ссылку.
7. Сброс игрока выполняется командой `/reset USER_ID`; после сброса у игрока снова `$10` и новый лимит 3–5 игр.

## Настройка Cloudflare D1 в браузере

1. Cloudflare Dashboard → Workers & Pages → D1 SQL Database → Create database.
2. Создайте базу, например `mines-db`.
3. Откройте ваш Pages-проект → Settings → Bindings → Add binding → D1 database.
4. Binding name должен быть строго: `DB`.
5. Выберите созданную D1 базу и сохраните.
6. В Variables добавьте:
   - `PARTNER_URL` — ваша партнёрская ссылка;
   - `TELEGRAM_BOT_TOKEN` — токен бота от BotFather;
   - `ADMIN_IDS` — Telegram ID админов через запятую, например `123456789,987654321`;
   - `SETUP_SECRET` — любой секрет для безопасной установки webhook, например `my-secret-123`.
7. Сделайте redeploy Pages-проекта.

Схема таблицы создаётся автоматически при первом API-запросе к `/api/player`, `/api/track`, `/bot` или `/set-webhook`.

## Настройка Telegram webhook

После деплоя откройте в браузере:

```text
https://ВАШ-ДОМЕН/set-webhook?secret=ВАШ_SETUP_SECRET
```

В ответе должно быть `"ok": true`. После этого команда `/admin` будет работать в Telegram-боте для ID, указанных в `ADMIN_IDS`.

## Важные файлы

- `app.js` — клиентская логика игры и блокировка после демо.
- `functions/[[path]].js` — Cloudflare Pages Function, D1 API, Telegram webhook и админка.
- `README.md` — эта инструкция.

## Важно

Игрок сможет получить дожим-сообщение только если он ранее открыл бота/Telegram WebApp, и Telegram разрешает боту писать этому пользователю.
